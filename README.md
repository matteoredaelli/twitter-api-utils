twitter-api-utils
=================

##Usage

Add in your file $HOME/.lein/profiles.clj 

	{:user {:env { :twitter-app-consumer-key "mykey" :twitter-app-consumer-secret "my-secr" :twitter-user-access-token "my-tock" :twitter-user-access-token-secret "my-tock-secr" }}}
